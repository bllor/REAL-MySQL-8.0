ch2. 설치와 설정

1.mysql 5.7버전과 8.0버전의 차이
[1]사용자 인증 방식 변경
mysql5.7버전에서는 native authentication 인증 방식을 사용하지만
8.0버전에서는 caching sha-2 authentication방식을 사용한다.
[2]외래키 이름의 길이
8.0버전에서는 외래키 이름의 길이를 64자로 제한단다.
[3]group by에 사용된 정렬 옵션
8.0버전에서는 group by 뒤에 정렬이 되지 않는다.
[4]
파티션의 각 테이블스페이스를 공용 테이블스페이스에 저장할 수 없다.

2.글로벌 변수와 세션변수
글로벌 범위의 시스템 변수는 하나의 mysql 서버 인스턴스에서 전체적으로 영향을 미치는 시스템 변수를 의미하며,
주로 mysql 서버 자체에 관련된 설정일 때가 많다.

세션 범위의 시스템 변수는 mysql클라이언트가 mysql 서버에 접속할 때 기본으로 부여하는 옵션의 기본값을 제어한다.

동적변수는 변경즉시 반경이 되는 것이고, 정적 변수는 서버 기동 중 변경이 불가능하다.
동적변수 변경시 set 명령어를 사용하는데, 이는 서버를 재시작할 경우 초기화 된다.
만약 계속 유지하고 싶다면 set persist 명령어를 사용하면 된다.
